02 / the surface

Thirteen services. One coherent registry.

every service · grpc-first
independently scalable · independently failed

Per-tenant isolation, all the way down

Every row carries a tenant_id. Every query filters on it. PostgreSQL Row-Level Security is layered underneath as a second wall. Each tenant gets its own custom hostname, its own storage prefix, its own RabbitMQ routing key. Cross-tenant data simply cannot move.

guarantee // RLS + per-tenant storage prefix + mTLS SNI

mTLS everywhere, not just at the edge

Every internal gRPC call is mutually authenticated. cert-manager in production, openssl-via-init-container in dev. Private keys are chmod 600, owned by uid 65532 — the distroless nonroot. Compromise one service and you still get nowhere.

crypto // x509 · cert-manager · zero plaintext hops

Cosign signing, in the hot path

Sigstore-compatible ECDSA P-256 signing. Private keys never leave Vault — sign/verify is a transit-engine call. Notary v2 is on the roadmap.

sigstore // ecdsa-p256 · vault-transit

Pull-through proxy cache

Cache Docker Hub, GHCR, gcr.io, quay.io. Credentials AES-256-GCM at rest. SSRF-blocked against all private CIDRs at connection time.

upstream // aes-256-gcm · ssrf-hardened

Pluggable vulnerability scanner

External process JSON-RPC. Trivy by default. Pin the binary checksum in env so a swapped scanner can't slip in.

contract // json-rpc · sha256-pinned

Immutable audit log

Append-only PostgreSQL partition. No UPDATE, no DELETE. Every push, every login, every scan — sealed.

retention // partitioned · append-only

Five storage drivers, one interface

MinIO for dev. AWS S3, GCS, Azure Blob, or plain filesystem for prod. The driver swap is one env variable.

drivers // minio · s3 · gcs · azure · fs

Webhook delivery you can trust

HMAC-signed bodies. Exponential backoff. Dead-letter routing. Tested against the receiver crashing mid-flight.

delivery // hmac-sha256 · retried · dlq

RBAC, all the way to the tag

Grant read, write, or admin at org, repo, or individual tag. Every grant and revoke lands in the immutable audit log. No "everyone-is-admin" fallback.

scope // org · repo · tag · audited

SSO with the providers you already run

Hand-rolled SAML 2.0 and OAuth 2.0 with PKCE. Google, GitHub, and Microsoft Entra wired out-of-the-box — or point it at any compliant IdP. JIT user provisioning, group → role mapping.

identity // saml · oauth+pkce · 3 idps

Observability you actually own

OpenTelemetry on every service, every span. OTLP straight to Jaeger, Grafana Tempo, Datadog, or stdout. Prometheus metrics on a dedicated internal port. Structured JSON logs with trace_id on every line.

exporters // jaeger · tempo · datadog · otlp

04 / the threat model

Security isn't a setting.
It's the build manifest.

govulncheck · gosec · gitleaks · in CI
all medium+ findings resolved
SEC · 001

RS256 JWTs, JTI revocation, 5-min TTL

Tokens are short-lived. Every validation hits Redis to confirm the JTI hasn't been revoked. Key rotation is zero-downtime via JWKS.

verifier: sha256:9a3f…b21c
SEC · 002

Argon2id passwords, never plaintext

API keys are returned in plaintext exactly once — at creation. Stored as Argon2id with fixed, non-configurable parameters that cannot be weakened from config: t=3, m=64 MiB, p=2.

kdf: argon2id · m=65536 · t=3 · p=2
SEC · 003

AES-256-GCM for upstream credentials

Pull-through proxy passwords are encrypted at rest with a key loaded from env at startup. Never logged, never database-stored in cleartext.

cipher: aes-256-gcm · key from env
SEC · 004

SSRF firewall on upstream registries

Every upstream URL is checked against private/loopback CIDRs at registration and on every connection — RFC 1918, link-local, CGNAT, IPv6 ULA all blocked.

cidr-deny: 10/8 · 172.16/12 · 192.168/16 · ::1
SEC · 005

Frontend token in memory, never localStorage

The React UI keeps bearer tokens in a Zustand store — process memory, tab-scoped, gone on refresh. Silent refresh 60s before expiry.

store: zustand · no-persist · 60s-refresh
SEC · 006

Distroless final images, uid 65532

gcr.io/distroless/static-debian12, nonroot, no shell, no package manager. The scanner runs on debian-slim — Trivy needs shared libs — and that exception is the only one.

base: distroless/static · 1 documented exception
SEC · 007

RBAC at org · repo · tag, audited

Every grant, every revoke, every check is a log line. The frontend calls GetUserPermissions on route load so the UI can't show actions the backend will reject.

scope: org · repo · tag · 3 roles
SEC · 008

SAML + OAuth, hand-rolled, audited

OAuth 2.0 with PKCE. SAML 2.0 with signature + assertion-encryption checks. Google / GitHub / Microsoft Entra preconfigured. No npm-package-of-the-day in the auth path.

protocol: saml2 · oauth2 · pkce
